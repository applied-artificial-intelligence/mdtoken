### Pre-commit Integration Flow

```
User commits file.md
       ↓
Pre-commit framework detects staged .md files
       ↓
Calls: mdtoken check file1.md file2.md ...
       ↓
Check command loads config (.mdtokenrc.yaml)
       ↓
LimitEnforcer checks each file
       ↓
Reporter displays results
       ↓
Returns exit code (0=pass, 1=fail)
       ↓
Pre-commit allows/blocks commit
```

## Key Implementation Details

### CLI Argument Handling
```python
@main.command()
@click.argument('files', nargs=-1, type=click.Path(exists=True))
@click.option('--config', default='.mdtokenrc.yaml')
@click.option('--dry-run', is_flag=True)
@click.option('--verbose', '-v', is_flag=True)
def check(files, config, dry_run, verbose):
    exit_code = check_files(
        files=list(files) if files else None,
        config_path=config,
        dry_run=dry_run,
        verbose=verbose,
    )
    sys.exit(exit_code)
```

### Pre-commit Hook Definition
```yaml
- id: markdown-token-limit
  name: Markdown Token Limit
  description: Check markdown files against token count limits
  entry: mdtoken check
  language: python
  files: \.md$
  pass_filenames: true
  args: []
```

### Exit Code Logic
```python
def get_exit_code(self, result: EnforcementResult, fail_on_exceed: bool) -> int:
    if not fail_on_exceed:
        return 0  # Always pass in dry-run mode
    return 1 if result.violations else 0
```
